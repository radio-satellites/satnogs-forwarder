# satnogs-forwarder
A down-to-earth satnogs forwarder

This connects to SoundModem, then forwards frames to the satnogs network!

Usage:

CALLSIGN NORAD ID LAT LONG

An example run would be

python main.py VE3SVF 53385 55.6932 22.3829
